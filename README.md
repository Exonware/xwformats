# xwformats
Enterprise serialization library with 18 formats (~87 MB): Protobuf, Avro, Parquet, Thrift, ORC, Cap'n Proto, FlatBuffers, HDF5, Feather, Zarr, NetCDF, MATLAB MAT, LMDB, GraphDB, LevelDB, BSON, UBJSON, XML. Separated from xwsystem to keep core lightweight (~5 MB, ~0.1s startup). Install only what you need. Part of eXonware.
